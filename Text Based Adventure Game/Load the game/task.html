Description

In this stage, you are going to implement the loading feature in your game. Don't forget to add Level 2! It's up to you to write this part of your story, but as a starting point for the JSON structure, you can copy-paste the Level 1 JSON elements in story.json to set up Level 2 and change the values from there.

We already included the loading option in our menu. Now it's time to polish it.

First, you need to check how many users are stored in the save folder. Show the list of the usernames without the %.json% extension and take the user input. Read the save file. If there are no save files or the user input contains an error, output the No save data found! message and go back to menu.

Once you loaded the save, you need to bring the game stats back into the game. After loading, the game should resume from where the player left off the last time.

Implement Level 2 according to the gameplay conditions that have been given to you in Stage 3.

When the player beats the game, output the Congratulations! You beat the game! message before finishing the program.

Objectives

In this stage, your program should:

  1. Show the list of usernames with saves.
  2. Take the user input to load the game. Use the prompt "Type your user name:" to signal the user to start typing.
  3. Output the respective error messages if something is wrong in the code.
